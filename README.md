# PPT Workflow Monorepo

这个仓库现在包含两个相关项目：

- `ppt-workflow/`：PPT 调研与页面效果图资产
- `html-slide-to-pptx/`：将结构化 HTML slide 转换为可编辑 PPTX 的工具

## 目录结构

```text
.
├── ppt-workflow/
└── html-slide-to-pptx/
```

## 子项目说明

### ppt-workflow

用于沉淀 PPT 调研结果、页面效果图和相关参考资料。

### html-slide-to-pptx

用于把结构化 HTML slide 转换为原生可编辑的 PowerPoint（.pptx）文件。

详细使用说明请分别查看各子目录内的 `README.md`。
